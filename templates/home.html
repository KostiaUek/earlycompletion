This is a webscraper that gathers info about a product listed on ceneo.pl based on it's id.

Purpose of the Application

Application works in the following way: based on the product id provided by a user, it starts scraping the products page (extracting the reviews). What it does is: checks how many reviews need to be extracted, and starts extracting them page by page, until the last one is parsed. After that, it saves the formated reviews into the correspondant json file.

Libraries Used

flask, socketio, mathplotlib, threading, numpy, io, glob, os, pandas, requests, json and bs4

Applied Approaches

The main applied approach of this application is to easily determine if it is worth to buy a certain product by having a complete overview of its reviews, including statistics and graphs based on them.
